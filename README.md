## Vue-IKnow 问答圈
用vue结合element-UI开发类似知加和小密圈的一款SPA应用

## 工具&技能

`vue`+`vuex`+`vue-router`+`vue-axios`

[barryvdh/laravel-cors](https://github.com/barryvdh/laravel-cors) 一个http代理的中间件，进行http请求转发，实现跨域请求

`postman` 接口测试工具
